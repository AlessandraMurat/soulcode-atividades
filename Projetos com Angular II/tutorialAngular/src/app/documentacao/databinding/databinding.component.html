Data binding

É a maneira de fazer vinculações entre componentes e o template, deste modo podemos dizer que a vinculção de dados é passada de um componente para o template e vice-versa.

Temos quatro tipos de data binding, sendo eles a interpolation, property binding, event binding e two way data binding

Interpolation

Permite incorporar valores de string dinâmicos em seu HTML. Use a interpolação para alterar o que aparece na visualização como a exbição do resultado de uma expressão

Interpolação refere-se a inclusao de expressões em texto marcado, por padrão usa-se chaves duplas {{ '{' }}{{ '{' }}{{ '}' }}{{ '}' }} como demilitadores. NO exemplo abaixo iremos mostrar o resultado de 5*3

<p> O resultado de 5*3 é: {{ '{' }}{{ '{' }} 5*3:{{ '}' }}{{ '}' }} </p>

E o resultado será:

O resultado de 5*3 é: 15

Property Binding

Os atributos de vinculação no Angular podem ajudá-lo a definir valores para atributos de elemento ou diretivas HTML. Use a associação de propriedade para realizar coisas como alternar funções de botão, definir caminhos de forma programática e compartilhar valores entre componentes.

Event Binding

A correlação de eventos permite que você ouça e responda às ações do usuário, como pressionamentos de tecla, movimentos do mouse e cliques

Para vincular a eventos, use a sintaxe de vinculação de evento Angular. Essa sintaxe consiste no nome do evento de destino entre parênteses no lado esquerdo do sinal de igual e na instrução do modelo entre aspas à direita. No exemplo a seguir, o nome do evento de destino é click e a instrução do template é Salvar().

<button> (click)= "Salver()" </button>

Two Way Data Binding

A vinculação bidirecional fornece uma maneira para os componentes do seu aplicativo compartilharem dados. Use a vinculação bidirecional para monitorar eventos e atualizar valores ao mesmo tempo entre os componentes.
