O que são Componentes ?

Os componentes são os principais blocos de construção das aplicações angulares. Cada componente inclui:

  • Um modelo HTML que renderiza o conteúdo da página;
  • Uma classe Typescrip que define o comotamento;
  • Um seletor CSS que declara como como o componente é usado no modelo;
  • Em opcional temos styles CSS que podem ser aplicados no modelo;

Dividindo o componente em duas partes

A primeira parte é Metadata que são definições que precisamos informar ao Angular que a nossa classe é um component, através de um decorator, no caso do component usamos o @Component.

A segunda parte é a classe, assim como outras lingaugens de programação que utiliza orientação a objeto , possui métodos e propriedades que serão listadas abaixo:

  • Selector: Responsável pela identificação do componente, o mesmo pode ser adicionado em um documento HTML
  • TemplateUrl: Conhecido como o nome do documento HTML que será a parte visual do componente. Nesta propriedade podemos inserir o código em HTML, juntamente com as diretivas e bindings necessários para exibição do componente no navegador.
  • Template: Outra alternativa para descrever a parte visual do componente.
  • StyleUrl: Aqui informamos a folha de estilo que compõem o componente

Portanto a estrutura de um componente deve ser formada por pelo menos 3 elementos, sendo eles o HTML, CSS e Typescript. Lembrando que os mesmos são organizados em arquivos se sepaardos e juntados através do decorator @Component.

Criando um componente através do CLI

Para a criação de um novo componente é importante já ter criado um projeto com o comando ng new, e acessar o repositório que foi criado, após isso podemos executar o comando a seguir:

1   |   ng generate component [nome do projeto]/[nome do componente]
