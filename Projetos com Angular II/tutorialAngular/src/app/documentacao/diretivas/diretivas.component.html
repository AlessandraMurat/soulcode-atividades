O que são diretivas ?

São classes que inclui comportamentos adicional aos elementos da nossa aplicação. São utilizadas em listas, estilos, formulários entre outros.

Existem três tipos de diretivas, sendo elas:

  • Diretivas integradas: Esse é o tipo mais comum;
  • Diretivas de atributo: Que alteram comportamento ou aparência de um elemento ou componente;
  • Diretivas estruturais: Que alteram o DOM adicionando ou removendo elementos;

Diretivas de atributo

São responsáveis por alterar o comportamento de um elemento, propriedade, componentes, atributos do HTML.

As mais comuns que são definidas pelo RouterModule e FormsModule, são:

  • NgModel: Adiciona vinculação de dados a um elemento de formulário HTML;
  • NgClass: Remove ou adicona um conjunto de classes CSS;
  • NgStyle: Adiciona ou remove um conjunto de estilos HTML.

Diretivas estruturais

são usadas para moldar ou remoldar a estrutura do DOM , geralmente adiciona, remove e manioula os elementos do host aos quais estão anexados

As diretivas estruturais mais comum são:

  • NgFor: Repetir um nó para cada item de um array;
  • NgIf: Condicional que cria ou discarta uma subvisualização de um modelo;;
  • NgSwitch: AConjunto de diretivas que alternam a visualização.
