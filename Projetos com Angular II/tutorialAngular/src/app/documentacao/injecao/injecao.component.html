O que é injeção de dependência ?

São serviços ou objetos que uma classe necessita para realizar sua função, a injeção de deped~encia(DI) é um modelo de desingn no qual uma classe solicita depedências de fontes externas para que não precise criá-las.

Criando um serviço

Para a criação de um serviço usamos o comando abaixo:

1   |   ng generate service [nome do projeto]/[service]

Serviços de injeção

A injeção de serviço tem como resultado torná-los disponíveis para os componentes.

Para injetar uam dependência em um componente, passamos para o constructor como argumento o tipo de depedência. Abaixo exemplificando temos que exemploService é do tipo ExemploService(Serviço criado através do ng generate service).

1   |   constructor(exemploService: ExemploService)
