O que são pipes ?

Pipe é a forma mais elegante que podemos alterar strings, valores monetários, datas para exibição. Pipes são funções simples que aceitam um valor de entrada e retorna uma sa´da editada. A vantagem de pipes é que pode ser usada em toda aplicação , enquanto declara cada pipe uma única vez. Abaixo veremos os tipos de pipes que já estão integrados ao Angular:

  • DatePipe: Edita um valor de data de acordo com as regras locais.
  • UpperCasePipe: Transforma o texto em letras maiúscula;
  • LowerCasePipe: Transforma o texto em letras minúscula;
  • CurrencyPipe: Transforma um número em string de moeda;
  • DecimalPipe: Transforma um número inteiro em uma string com um ponto decimal;
  • PercentPipe: Transforma um número em uma string de porcentagem;
  • SlicePipe: Cria um novo Array ou String contendo um subconjunto dos elementos.

Aplicando pipe em um elemento

Para usar um pipe usamos o operador de pipe (|) em uma expressão de um modelo. A seguir iremos usar um modelo com o nome de caixaAlta que receberá uma string no arquivo teste.component.ts

import {{ '{' }} Component{{ '}' }} from '@angular/core';

@NgComponent({{ '{' }}

    selector:['app-teste'],

    template: ['./teste.coponent.html']

{{ '}' }})

export class TesteModule {{ '{' }} caixaAlta: 'Testando caixa alta' {{ '}' }}

No arquivo teste.component.html vamos inserir uma tag p e inserir o pipe

<p> Teste {{ '{' }} caixaAlta | uppercase {{ '}' }} </p>
