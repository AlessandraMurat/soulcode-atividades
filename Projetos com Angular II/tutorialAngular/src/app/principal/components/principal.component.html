O que é Angular?

É um framework de código aberto mantido pela Google para a criação de Aplicações de Página Única (SPA).

Tendo como finalidade deixar o desenvolvimento de aplicação SPA mais simples e otimizado. Podemos desenvolver aplicações para mobile e desktop, tornando o processo modernos e escaláveis.
