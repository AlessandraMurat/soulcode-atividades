Angular CLI

O que é CLI ?

A equipe do angular vendo a dificuldade de inciar e gerenciar um novo projeto criaram uma ferramenta de limnha de comando, a Interface de Linha de Comando - Command Line Interface (CLI), tendo como objetivo facilitar o gerenciamento de projetos escritos nesse framework.

Instalando o Angular CLI

Para utilizar o Angular CLI é necessário instlá-lo, lembrando que é primordial que você já tenha feito a instalaçãp do Node.JS.

Usaremos o comando npm, necessário para a instalaçãom do Angualr CLI. Vamos para o primeiro comando, para isto abra seu prompt de comando.

1   |   npm install -g @angular/cli

Após dar enter é só aguardar, pois a instalação pode demorar em tempo considerável até que a instalação esteja concluída. Assim que a instalação terminar já podemos utilizar o Angular Cli através do comando ng, que é um programa pasara ser executado pela linha de comando e está pronto para iniciar um novo projeto.

Criando um novo projeto

Para criar um novo projeto em angular iremos usar o comando ng new que ao ser executado criará um diretório com uma estrutura base, co arquivos necessários para iniciar uma aplicação.

1   |   ng new [nomeProjeto]

Após a criação do diretório podemos visualizar no servidor todas as alterações realizadas nos arquivos da aplicação, basta utilizar o comando abaixo:

1   |   npm install -g @angular/cli

Por padrão o servidor criado fica "escutando" na porta 4200, basta inserir o seguinte caminha no seu navegador localhost:4200, a visualização da sua página será como a da imagem ao lado.
